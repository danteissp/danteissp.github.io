¿Que es HTML?

Actualmente la mayoria de las personas han visitado muchas paginas web pero muchas siguen preguntandose ¿Que es HTML?

Si buscas en la internet la definicion de HTML, probablemente encuentraras esta: "HTML es un lenguaje de marcas de hipertexto que se utiliza para el desarrolo de paginas web. Sus siglas en ingles son de HyperText Markup Languague".

¿Para que sirve?

HTML sirve para codificar la estructura de una pagina web. Para ello se usa una de sus caracteristicas llamada etiquetas o en ingles tags. Los navegadores no muestran las etiquetas pero las utilizan para representar el contenido de la pagina.
